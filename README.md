# Graduate Kanban

Synced Apps Script deployment:

The Apps Script version stores tasks in a Google Sheet created on first use. GitHub Pages can still serve the static app, but that version only uses browser local storage.

## Commands

```bash
npm run dev
npm run build
npm run apps:build
npm run apps:push
npm run apps:deploy
```
